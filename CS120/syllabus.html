二、课程简介和教学目的

This course is an introduction to computer networks. In addition to conceptual knowledge (Ethernet, switching, routing, IP, TCP, email, etc.), this course will show you how to build a real computer network from bottom up. To this end, the course contains a project, which includes four subprojects. They will guide you to build a toy network through the audio interface in your computers.
