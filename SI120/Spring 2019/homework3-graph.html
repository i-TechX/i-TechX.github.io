Problem 2
?/? point (graded)
We have three buckets A, B, C of 5, 3 and 2 litres respectively. At the beginning the bucket A is full and the others are empty. We allow the operation of pouring the bucket X in the bucket Y until X is empty or Y full. Can we do this until the bucket A contains 2 litres, the bucket B contains 2 litres too, and the bucket C contains 1 litre? Hint: Draw the oriented graph such that vertices are triples (a, b, c) satisfying a + b + c = 5 and an edge from one vertex to another one when we can perform the pouring operation from the first vertex to the other one. Eg: there is an edge from (5, 0, 0) to (3, 0, 2).

Problem 5
?/? point (graded)
Assume the rectangle below represents the map of a zoo, with several areas delimited by fences (represented by segments in the rectangle). To organize the visit of the zoo, we want to find a route going through all the areas.

(a) Can we find a route crossing each fence exactly once, considering that we cannot cross the external fences (boundary of the big rectangle)? If yes, draw it.

(b) Can we find a route crossing each fence exactly once, considering that we also have to cross all the external fences exactly once too? If yes, draw it.

(c) Assume that the entrance/exit of the zoo is in the top leftmost area. Can we organize a route starting here, going through all the areas exactly once and coming back to the entrance/exit? If yes, draw it.
